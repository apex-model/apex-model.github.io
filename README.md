# apex-model.github.io

Landing page for the APEX modeling ecosystem, served at
<https://apex-model.github.io>.

It is a static gateway that points visitors to the right project and its
documentation:

- **APEX** -- the process model ([repo](https://github.com/apex-model/apex))
- **QAPEX** -- the QGIS interface ([repo](https://github.com/apex-model/qapex))
- **APEX Toolbox** -- desktop toolbox (coming soon)

Plain HTML, no build step. GitHub Pages serves `index.html` from the default
branch.
